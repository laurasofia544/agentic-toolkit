Agentic Orchestration Toolkit

Overview

This CLI toolkit integrates:
  • OpenAI Web Search
  • OpenAI Image Generation
  • Playwright Screenshot Automation
  • Gemini Multimodal Design Feedback

Commands

Web Search:

npm run dev -- web-search "latest AI news"

Image Generate:

npm run dev -- image-generate "A cinematic lighthouse in a storm"

Design Feedback:

npm run dev -- design-feedback "https://example.com"

Output Structure

references/
  web-search__*.md
  screenshots/
  aI_feedback/

images/

Agentic Development Process

This project was developed by orchestrating an AI coding agent to:
  • Design CLI architecture
  • Integrate OpenAI web search tool
  • Implement image generation
  • Configure Playwright automation

Technical Stack

Node.js • TypeScript • OpenAI API • Gemini API • Playwright • Commander
